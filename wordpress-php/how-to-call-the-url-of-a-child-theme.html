How To Call The URL Of A Child Theme

In your functions.php, add in this code

<?php

function get_template_directory_child() {
    $directory_template = get_template_directory_uri();
    $directory_child = str_replace('twentyseventeen', '', $directory_template) . 'YOUR CHILD THEME';
    return $directory_child;
}

?>

Where ever you want to call the url add 

<?php echo get_template_directory_child() ?>
